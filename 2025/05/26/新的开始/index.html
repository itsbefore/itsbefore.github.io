信息搜集

信息搜集其实是⼀个⾮常⼤的块，⼀时半会讲也讲不完，不过⽬前来说最基本的就⼏个：

  1. 域名信息搜集
  2. 端⼝扫描
  3. github等敏感信息泄漏

⽬前来看初步定在⼀个arl灯塔来做基础的信息平台，oneforall等脚本做补充的域名信息搜集。

arl灯塔已经删库停止维护了，这里意外找到了一个新的安装方法：

本地docker安装ARL灯塔

1、安装docker docker-compose

2、导入所需要的文件：

通过网盘分享的文件：灯塔

3、下载并导入文件，解压ARL-2.6.1.zip，可使用unzip解压

unzip ARL-2.6.1.zip

4、docker导入镜像

docker load <arl_2.6.1.tar

docker load < arl_mongo.tar

docker load < arl_rabbitmq.tar

使用docker images 查看

5、vim ARL-2.6.1目录下/docker/.env

cd ARL-2.6.1/docker/

vim .env

修改ARL_VERSION=v2.6.1

保存退出

6、首次部署需要执行docker volume create arl_db

7.在/docker/目录下，docker-compose up -d（或docker compose up -d）启动

8、访问地址IP:5003

账号：admin

密码：arlpass

（记得修改密码）
